vimpager-perlmod
================
Script to use VIM as a PAGER.

Custom syntax file for vim to use with perldoc documents.

Place *examples/vimpagerrc* in $HOME/.vimpagerrc or $XDG_CONFIG_HOME/vimpagerrc.
